### Keaton — C++/DOD Performance Audit

**Scope:** Audit of hot paths and idiomatic EnTT usage against DoD principles.

**Key Findings:**
- **Rule 1 — Per-frame component mutation:** Use `get_or_emplace<T>(e) = value`, not `emplace_or_replace<T>(e, value)` (unnecessary signal dispatch). Violation: `camera_system.cpp:373` ScreenPosition (fix mirrors issue #244, already resolved for ModelTransform).
- **Rule 2 — Branching inside views:** Use structural views (separate Miss/Hit views), not `any_of<T>` inside a shared view. Violation: `scoring_system.cpp:36` branches on MissTag inside shared view (low cardinality, pattern debt only).
- **Rule 3 — Hoist ctx() lookups:** Move `reg.ctx().find<T>()` outside loops. Violations: `scoring_system.cpp` (2×), `collision_system.cpp`, `miss_detection_system.cpp`.
- **Out-of-scope deferred:** UI JSON per-frame (high refactor cost), BurnoutState dead code (cross-team sign-off needed), lifetime_system static vector (quick win), obstacle_spawn RNGState lazy emplace (minor).

**Owner:** McManus (primary), Keaton (optimizations).

---

### Kujan — Audit Review and Materiality Assessment

**Scope:** Synthesis and prioritization of Keyser and Keaton findings; noise filtering.

**Material Findings:**
- **F1 (MEDIUM):** `scoring_system` removes iterated view components inside loop (latent UB trap — currently safe but fragile). Fix: collect-then-process pattern (mirror cleanup_system).
- **F2 (MEDIUM):** `COLLISION_MARGIN` tripled across 3 files (consistency risk). Fix: promote to `app/constants.h`.
- **F3 (MEDIUM):** `APPROACH_DIST` duplicated in `constants.h` and `song_state.h`. Fix: deduplicate, use `constants::APPROACH_DIST`.
- **F4 (LOW-MEDIUM):** System logic in component headers (ensure_active_tags_synced, wire_obstacle_counter). Fix: move to system `.cpp` files.
- **F5 (LOW):** Render systems take non-const `entt::registry&` (forfeits compile-time safety). Fix: make signatures `const`.
- **F6 (LOW):** Component structs with non-trivial methods (HighScoreState, SettingsState). Fix: extract mutation logic to systems.

**Noise:** cleanup_system static buffer (intentional #242), no groups (correct choice), BurnoutState stale fields (no UB).

**Owner:** McManus (primary fixes), Fenster (constants).
